# loan-advisor-ai
An intelligent loan default prediction and explainable AI system built using a stacking ensemble of multiple models on the Home Equity (HMEQ) dataset.  
Includes a LangChain RAG chatbot and a React-based frontend for interactive financial insights.

## Features
- Stacking classifier combining Random Forest, Decision Tree, XGBoost, Logistic Regression, Naive Bayes, SVM, and Gradient Boosting  
- Target variable **BAD** indicates whether the applicant defaulted on loan payments  
- Explainable AI with SHAP and LIME  
- Financial assistant chatbot powered by LangChain RAG  
- **Frontend:** React for interactive user interface  
- **Backend:** FastAPI for model serving and chatbot API integration

## Tech Stack
Python · Scikit-learn · XGBoost · SHAP · LIME · FastAPI · LangChain · React
